# WAD2022-hw4 Backend


## setting up database to test
Start up a localhost Postgre database with name "testwad" and make sure to add
your password to the database.js file


## Navigate to the project directory
```
cd <project name>
```

## Install dependencies
```
npm install
```

## Run the project
```
npm start
```

or 
```
npm server.js
```
